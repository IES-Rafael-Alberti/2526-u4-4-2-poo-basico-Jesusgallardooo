## Descripción de la Actividad
Actividades de POO para empezar a familiarizarnos con clases y objetos en Kotlin. Vamos a crear clases con atributos privados,
aplicar validaciones en los datos mediante setters o init, sobrescribir toString() para mostrar la información de los objetos,
y probarlos en un programa principal para asegurarnos de que todo funciona correctamente.
